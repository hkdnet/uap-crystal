# user_agent_parser

Simple and useful lib for parsing user agent string.

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  user_agent_parser:
    github: hkdnet/user_agent_parser
```


## Usage


```crystal
require "user_agent_parser"
```
